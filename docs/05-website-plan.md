# Website Plan

## 1. Purpose

The Website is the public, SEO-first web presence of MyClup.

Goals:

- Explain the product clearly
- Convert gyms into leads or customers
- Support discovery of gyms
- Publish legal and informational pages
- Rank well in search engines
- Operate as a multilingual, SEO-aware public surface from day one

## 2. Audience

- Gym owners
- Instructors
- Members
- Prospective members
- People searching for gyms nearby
- Search engines

## 3. Website Sections

The website must be multilingual from the start, not treated as a later enhancement.

### 3.1 Marketing Pages

- Homepage
- Features
- Solutions by role
- Pricing
- How it works
- Chat-first communication page
- Gym management page
- Instructor tools page
- Member app page

### 3.2 Conversion Pages

- Book demo
- Contact sales
- Start free trial
- Request onboarding

### 3.3 Discovery Pages

- City pages
- District pages
- “Gyms near me” pages
- Gym profile pages
- Category pages: Pilates, CrossFit, Yoga, PT, Boxing

### 3.4 Trust and Legal

- About
- Privacy policy

- Cookie policy
- KVKK disclosure
- Security page
- Support page

### 3.5 Content / SEO

- Blog
- Guides for gym owners
- Fitness and class booking guides
- Local landing pages
- Comparison pages
- Feature release pages

## 4. SEO Strategy

- Strong information architecture
- Programmatic landing pages for locations and gym categories
- Schema markup
- Fast performance and Core Web Vitals
- Unique copy per important page
- Conversion-focused metadata
- Internal linking between discovery and feature pages
- Locale-aware routing
- `hreflang`, canonical, and alternate locale handling
- Localized metadata, structured data, and sitemap generation

## 5. Discovery Marketplace Integration

- Public gym listings
- Filters
- Map support later
- Trial class CTA
- Contact or chat CTA
- Review snippets
- Opening hours
- Amenities
- Instructor highlights

## 6. Conversion Features

- Sticky CTA for demo
- Interactive product tours
- ROI calculator for gyms
- Case studies
- Testimonials
- Comparison tables
- FAQ sections
- Locale-specific CTAs and translated lead forms

## 7. Analytics and Growth

- SEO traffic
- Demo conversion rate
- Trial conversion rate
- Discovery page CTR
- Gym profile lead rate
- Content performance
- Performance by locale

## 8. MVP Scope

- Homepage
- Features
- Pricing
- Contact / demo
- Legal pages
- Basic gym profile template
- Basic SEO setup
- Multilingual routing and locale-aware SEO foundations

## 9. Future Enhancements

- Interactive live product preview
- Full local SEO network
- CMS-backed publishing workflow
- A/B testing

## 10. Localization Requirements

- Support locale-prefixed routing
- Store legal and marketing pages per locale
- Support translated gym listing content where available
- Fallback safely to a default locale when translation is missing
- Translate navigation, metadata, forms, and shared website UI consistently

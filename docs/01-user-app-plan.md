# User App Plan

## 1. Purpose

The User App is the member-facing Expo mobile application.

Primary goals:

- Let members track memberships easily
- Let members book and manage classes
- Let members chat with gyms and instructors effortlessly
- Let members monitor fitness history and progress
- Let members discover new gyms nearby
- Let members use the entire app in their preferred language

## 2. Primary User Types

- Active gym member
- PT client
- Group class member
- Trial user
- Prospect discovering gyms

## 3. Core App Sections

### 3.1 Home

- Membership status card
- Upcoming class / appointment widget
- Outstanding payment widget
- Latest messages widget
- Suggested nearby gyms
- Promotions and announcements
- Quick actions: Book class, Open chat, Show QR, Renew membership

### 3.2 Chat

This should be one of the main tabs and the strongest experience in the app.

- Conversation list
- Search
- Unread badge counts
- 1:1 with gym
- 1:1 with instructor
- Group class chats
- Media and voice note support
- Read receipts
- Typing indicators
- Quick reply suggestions
- Attachments: images, PDFs, workout sheets
- Pinned conversations
- Conversation mute / archive

### 3.3 Membership

- Current package details
- Validity dates
- Remaining sessions / credits
- Frozen status
- Renewal options
- Payment history
- Invoice view
- Add-ons
- Branch access rights
- QR / barcode member pass

### 3.4 Classes and Appointments

- Weekly calendar view
- Daily schedule view
- Class search and filters
- Book a class
- Cancel booking
- Join waitlist
- Appointment reminder settings
- Rebook previous session
- Show instructor details
- Show available spots

### 3.5 Progress

- Body measurements timeline
- Weight history
- Attendance streaks
- Workout history
- Completed classes
- Personal records
- Progress photos
- Instructor notes visible to member
- AI-generated summaries of progress trends

### 3.6 Discover

- Nearby gyms map view
- Nearby gyms list view
- Filter by distance, price, category, amenities
- Gym profile pages
- Reviews and ratings
- Trial class / visit request
- Compare gyms
- Favorite gyms
- Recommended gyms based on interests and location

### 3.7 Profile

- Personal info
- Health notes
- Emergency contact
- Notification settings
- Privacy settings
- Language settings
- Linked gyms
- Membership agreements
- Support / help center

## 4. Key Flows

### 4.1 New Member Onboarding

- Phone or email sign up
- Profile completion
- Join an existing gym account via invite / member ID
- Accept waivers and consent
- Enable notifications

### 4.2 Booking a Class

- Discover class from calendar or recommendations
- View class details
- Book instantly
- Receive confirmation
- Add to personal calendar
- Receive reminder before class

### 4.3 Membership Renewal

- Receive renewal reminder
- See available packages
- Pay in app or submit request
- Get updated access instantly

### 4.4 Chat-Based Support

- Open support chat
- Pick quick topic: Membership, payment, schedule, coach, technical issue
- Continue as natural conversation
- Receive attachments and links

### 4.5 Localization Experience

- Detect device locale on first launch
- Allow language selection from settings
- Respect stored user language preference across sessions
- Localize dates, times, numbers, and currencies
- Show localized membership, billing, and legal text when available
- Prefer the member's selected language in campaigns and support templates

## 5. High-Value Features

- Fast QR-based check-in
- Apple Wallet / Google Wallet card later
- Voice message to instructor
- Class reminder with one-tap cancel
- Smart nudges for inactive members
- Anniversary / achievement badges
- Member referral system
- Loyalty points
- Freeze request from app
- Transfer package request
- Gift membership request

## 6. AI Features

- AI explains workout plans in simpler language
- AI summarizes coach notes
- AI recommends classes based on attendance patterns
- AI suggests best booking times
- AI surfaces progress milestones
- AI can explain outputs in the member's selected language

## 7. Notifications Strategy

- New message
- Membership expiring soon
- Class reminder
- Waitlist promoted
- Payment due
- Progress milestone
- Nearby gym promotion
- Campaign / offer

## 8. UX Principles

- Chat must feel as easy as WhatsApp
- Critical actions within 1 to 2 taps
- Calendar interactions should be fluid
- Member history must be easy to browse
- Discovery should feel inspiring, not crowded

## 9. Metrics

- DAU / WAU
- Chat opens per user
- Booking conversion rate
- Renewal rate
- Notification open rate
- Discovery to trial conversion
- Member retention
- Locale adoption and translation quality coverage

## 10. MVP Scope

- Auth
- Profile
- Membership overview
- Class booking
- Basic chat
- Notifications
- Gym discovery basics
- Full client-side localization foundation
